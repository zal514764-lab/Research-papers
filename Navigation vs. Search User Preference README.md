Navigation vs. Search User Preference
Description
Empirical investigation of whether users prefer navigating information hierarchies (folders, menus) over issuing search queries or browsing when retrieving digital information, synthesizing evidence from personal information management (PIM) and web contexts.

Technologies
Google Scholar / ACM Digital Library literature search

Log analysis (file retrieval events)

Dual-task cognitive load experiments

fMRI (functional magnetic resonance imaging)

Survey and field study methodologies

Study Types Evaluated
Lab experiments (controlled file retrieval tasks)

Field studies (real-world usage logs)

Neurocognitive studies (fMRI, attention resource measurement)

Survey/interview studies (user preference self-reports)

Results
Study	Context	Navigation Use	Search Use	Key Finding
Bergman et al. (2008)	Personal files	56–68%	4–15%	Search = last resort
Israeli et al. (2019)	Shared files	84%	4%	Navigation dominates all ages
Bergman & Dvir (2024)	Personal files	~95% (control)	~5%	Only disorganized users search more
Bergman et al. (2013)	File retrieval	3× faster	3× longer	Navigation = lower cognitive load
Alkindi & Bouazza (2010)	Academic websites	60–90%	—	Users prefer browsing menus
Key Concepts
Navigation preference: Users choose folder/menu traversal over keyword search even when search is available

Cognitive load explanation: Navigation consumes fewer attention resources (validated via dual-task and fMRI)

Spatial memory activation: Folder navigation uses posterior hippocampal regions (same as real-world navigation); search activates Broca's area (language)

Search as "last resort": Users switch to search only when file/folder location is forgotten

Hyper-searchers: Small minority (~5%) who search due to disorganized filing, not preference

Short Summary
Across eight empirical studies (2008–2024), users consistently prefer navigation over search for file retrieval (56–95% of retrievals) and browsing over search for websites (60–90%). Neurocognitive evidence shows navigation requires less cognitive load and activates spatial memory brain regions, making it feel more "automatic." Search is used primarily as a last resort when users cannot remember where an item is located. The preference persists even with improved search technology and across age groups (though older users search slightly more). Exceptions include exploratory tasks or highly disorganized personal filing systems.
